配置示例

简洁的YAML配置，强大的功能支持

config.yaml 
Settings:
  max_history: 7  # 保留最近7天文章
  load_images: true  # 下载并嵌入图片
  filename_template: "RSS精选_{date}.epub"  # 自定义文件名

Feeds:
  - url: "https://sspai.com/feed"
    name: "少数派"
    enabled: true
    resolve_link:
      enabled: true
      method: "readability"  # 智能全文提取
      
  - url: "https://feeds.feedburner.com/ruanyifeng"
    name: "阮一峰"
    enabled: true

常见问题

快速了解产品核心信息

需要付费吗？

完全免费！基于MIT开源协议，您可以自由使用、修改和分发。

支持哪些RSS源？

支持所有标准RSS/Atom格式的订阅源，可同时配置多个源进行聚合。

如何设置自动推送？

使用GitHub Actions，只需配置一次即可实现每日定时自动推送，也支持手动触发。

支持哪些Kindle设备？

支持所有能接收邮件推送的Kindle设备，包括Kindle Oasis、Paperwhite、基础版等。

图片能正常显示吗？

支持图片自动下载和嵌入，WebP格式会自动转换为Kindle兼容格式，并防止跨页显示。
