Configuration Example

Simple YAML configuration, powerful feature support

config.yaml 
Settings:
  max_history: 7  # Keep articles from last 7 days
  load_images: true  # Download and embed images
  filename_template: "RSS_Digest_{date}.epub"  # Custom filename

Feeds:
  - url: "https://example.com/feed"
    name: "Tech Blog"
    enabled: true
    resolve_link:
      enabled: true
      method: "readability"  # Smart full-text extraction
      
  - url: "https://news.example.com/rss"
    name: "News Feed"
    enabled: true

Frequently Asked Questions

Quick understanding of core product information

Is it free?

Completely free! Open source under MIT license, you can freely use, modify and distribute.

Which RSS sources are supported?

Supports all standard RSS/Atom format feeds, can configure multiple sources for aggregation.

How to set up automatic push?

Using GitHub Actions, configure once for daily scheduled automatic push, also supports manual trigger.

Which Kindle devices are supported?

Supports all Kindle devices that can receive email push, including Kindle Oasis, Paperwhite, Basic, etc.

Can images display properly?

Supports automatic image download and embedding, WebP format auto-converts to Kindle-compatible format with anti-page-break display.
